 Retail Sales Performance Analysis (Advanced Excel)

 Project Overview
This project demonstrates an end-to-end **Excel-based data analytics workflow** using a retail sales dataset.  
It focuses on **data cleaning, analysis, automation, and dashboarding** aligned with Data Analyst job requirements.

---

## Tools & Skills Used
- Microsoft Excel (Advanced)
- Power Query (ETL)
- Pivot Tables & Pivot Charts
- Advanced Excel Formulas
- VBA & Macros
- Interactive Dashboard Design

---

## Dataset
- Source: Kaggle (Superstore Sales Dataset)
- Records include sales, profit, discounts, regions, and product categories

---

##  Project Workflow
1. Imported and cleaned data using **Power Query**
2. Applied **advanced Excel formulas** (XLOOKUP, AGGREGATE, SUBTOTAL, IFERROR)
3. Built **Pivot Tables and Pivot Charts**
4. Designed an **interactive Excel dashboard**
5. Automated refresh, reset, and export using **VBA macros**

---

##  Dashboard Features
- KPI Cards (Sales, Profit, Orders, Profit Margin)
- Region, Category, and Year slicers
- Dynamic charts and pivot tables
- One-click PDF export

---

## Automation
- One-click data refresh
- Dashboard reset using VBA
- Automated Power Query integration
